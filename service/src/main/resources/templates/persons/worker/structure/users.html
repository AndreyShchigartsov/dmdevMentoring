Инструкция по добавлению детей в отряд

  • Найдите нужного ребенка с помощью фильтров поиска
  • Нажмите кнопку "Добавить в отряд" на карточке ребенка
  • Выберите подходящий отряд из списка доступных
  • Подтвердите добавление ребенка в отряд
  • Ребенок будет автоматически зачислен в выбранный отряд
  • Один ребенок может находиться только в одном отряде одновременно
